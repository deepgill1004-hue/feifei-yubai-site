# 我可以自主落地的範圍

## 我現在可以直接做

1. 建立蘇菲餘白品牌網站 MVP。
2. 整理品牌定位、首頁文案、CTA 與閱讀路徑。
3. 產出電子報歡迎信、訂閱頁文案、方格子 CTA 模板。
4. 建立 LINE 關鍵字承接腳本。
5. 建立內容地圖、樞紐文規劃、舊文重包裝清單。
6. 把專案整理成可部署到 GitHub Pages 的靜態網站。
7. 在你指定 GitHub repo 後，幫你建立檔案、更新檔案或準備 commit。

## 我不會擅自做

1. 讀取瀏覽器登入狀態、密碼、cookie、API key。
2. 連到你的私人帳號後台操作。
3. 建立或刪除遠端 GitHub repo。
4. 發送正式電子報。
5. 發布文章或公開網站。
6. 上傳含個資、醫療紀錄、客戶資料的內容。

## 最優落地策略

第一階段不做大型 WordPress，也不假裝已註冊電子報平台。先完成：

方格子文章庫 → 蘇菲餘白品牌網站 → LINE → 電子報表單預留 → 諮詢

等名單、流量與轉換證明有效，再做 WordPress 內容站。
